### What I learned

In this project, I strengthened my CSS layout skills by learning how to perfectly center a component on the screen using CSS Flexbox on the `body` tag:

```css
body {
    min-height: 100vh;
    display: flex;
    justify-content: center;
    align-items: center;
}
