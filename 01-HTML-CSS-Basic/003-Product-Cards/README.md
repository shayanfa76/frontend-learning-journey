# Product Cards

This project focuses on building a clean, modern, and responsive e-commerce product card layout using pure HTML and CSS.

## 🎯 Project Objectives
- Practice CSS Flexbox for structured and responsive card alignment.
- Implement dynamic pricing layouts including discount badges and strike-through original prices.
- Use embedded inline SVGs for high-quality, scalable UI icons and star ratings.

## 💡 Key Learnings
In this project, I learned how to handle complex text wrapping inside flexible layouts, manage spacing using gap properties, and integrate custom Persian typography (Shabnam font) for a localized and user-friendly interface.
